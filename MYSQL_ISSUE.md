# Starting MySQL with Laragon

## Quick Steps

1. **Open Laragon**
   - Find Laragon in your Start Menu or taskbar
   - Or run: `C:\laragon\laragon.exe`

2. **Start MySQL**
   - In Laragon window, click **"Start All"** button
   - Or right-click Laragon tray icon → **Menu** → **MySQL** → **Start**

3. **Verify**
   - Laragon should show "MySQL" as **green/running**
   - You should see the Laragon tray icon active

## Verify Database

To confirm your database is there:
1. In Laragon, click **"Database"** button
2. This opens **HeidiSQL** or **phpMyAdmin**
3. Check that database `tunistudent` exists with your tables
